## Tailwind

### Pros 🤠
* Quick to prototype
* Atomic CSS

---
## Tailwind

### Cons 🤦‍♂
* No official React Native support
* Big bundle size (purgecss can help)
* CSS pollution
* Very hugeeee number of classes
* Extra build tools

---
## Styled Components

## Pros 🤠
* Official React Native support
* No CSS pollution
* Composables CSS with React Components
* Component Name is attached with the styles
* No extra build tools
---

## Styled Components

## Cons 🤦‍♂
* Write CSS by yourself
* Difficult to write atomic CSS
---

### Why can't we do this?

```jsx
const StyledContainer = styled.div`
    .text-blue-600; 
    .lg:text-green-600; 
    .hover:bg-yellow; 
    .pt-10; 
`
```
---

## Styled Wind 🎉

* Official React Native support
* Tailwind like atomic CSS
* Smaller bundle size
* Composables CSS with React Components
* Component Name is attached with the styles
* No extra build tools
